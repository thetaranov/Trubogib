# ПЛК Owen PLC110-30 — Программа трубогиба

## Описание

Программа для Owen PLC110-30 (Modbus), написанная в CoDeSys v2.3.9 (Structured Text).
Управляет осью гиба (ось B) через энкодер 360 P/R и дискретные выходы на двигатель.

## Файлы

| Файл | Описание |
|------|----------|
| `TRUBOGIB.pro` | Основной проект CoDeSys v2 |
| `TRUBOGIB.SDB` | Символьная база данных |
| `TRUBOGIB_CLEAN.exp` | Экспорт проекта (текстовый) |
| `export_codesys/` | Экспорт отдельных POU |
| `архив/` | Архивные версии проекта |

## Связи ПЛК

### Входы
| Адрес | Назначение |
|-------|------------|
| IX0.0 | Энкодер канал A (HSC, быстрый вход) |
| IX0.1 | Энкодер канал B (HSC, быстрый вход) |
| IX1.4 | Переключатель режима (D5, ручной/авто) |
| IX1.5 | Педаль «Вперёд» (D7) |
| IX1.6 | Педаль «Назад» (D8) |

### Выходы
| Адрес | Клемма | Назначение |
|-------|--------|------------|
| QX3.0.4 | DO9 | SON пара 1 — Servo ON ось Z (синий провод CN1) |
| QX3.0.5 | DO10 | Двигатель CW (гибка +) |
| QX3.0.6 | DO11 | Двигатель CCW (гибка −) |
| QX3.0.7 | DO12 | SON пара 2 — Servo ON ось C (синий провод CN1) |
| QX5.0 | — | Бипер (пищалка) |

### RS-485
- **COM1** — Modbus RTU Master
- Скорость: **38400**, **8N1** (8 бит, без чётности, 1 стоп-бит)
- Slave: Teensy 4.1 (ID=1)

## Установка CoDeSys

1. Установить CoDeSys v2.3.9.68 (patch 5)
2. Установить Target-файл: `PLC110.30_m_v2/IOModules/PLC110_30_v2.cfg`
3. Открыть `TRUBOGIB.pro`

## Прошивка

```
Online → Communication Parameters → COM-порт (USB-RS485)
Online → Login
Online → Download
Online → Run
```
