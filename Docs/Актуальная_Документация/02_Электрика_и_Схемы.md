## 4. Самодельный кабель управления CN1 (LAN)

Для связи макетной платы (шлюза Teensy) и каждого из сервоусилителей (управление осями Z и C) используется **самодельный кабель на базе стандартной витой пары (LAN)**. 
По этому кабелю передаются высокочастотные дифференциальные сигналы RS-422, сигналы управления реле и питание для изолированных входов.

### 4.1. Таблица распайки LAN → Разъем CN1 (Схема для одного кабеля)

Поскольку у нас 2 сервопривода (Ось Z и Ось C), мы используем **два отдельных LAN-кабеля** и **4 платы MAX490** (по 2 на ось: одна для Step, одна для Dir).

| Цвет жилы (LAN) | Пара | Пин CN1 | Сигнал | Функция | Куда идет на плате (Шкаф) |
|-----------------|------|---------|--------|---------|---------------------------|
| **Оранжевый** | 1 | **10** | PP | Step + (Импульс +) | MAX490 "Шаг" (выход Y) |
| **Бело-Оранжевый** | 1 | **11** | PG | Step - (Импульс -) | MAX490 "Шаг" (выход Z) |
| **Зеленый** | 2 | **35** | NP | Dir + (Направление +) | MAX490 "Направление" (выход Y) |
| **Бело-Зеленый** | 2 | **36** | NG | Dir - (Направление -) | MAX490 "Направление" (выход Z) |
| **Синий** | 3 | **15** | SON | Servo-ON (Включение) | На ПЛК DO7 (пара 1/Z) или DO8 (пара 2/C) — релейный sink (NPN) |
| **Бело-Синий** | 3 | **48** | ALM | Alarm (Ошибка) | На вход ПЛК / Teensy |
| **Коричневый** | 4 | **20** | DICOM | Плюс питания входов | **+24В** (от БП 60Вт) |
| **Бело-Коричневый** | 4 | **46** | DOCOM | Общий минус выходов | **0В (GND)** (от БП 60Вт) |
| **ЭКРАН (фольга)** | — | Корпус | Shield | Заземление | Металлический кожух CN1 / Шкаф |

### 4.2. Подключение MAX490 к Teensy 4.1 (Обе оси)

* **Ось Z (Кабель LAN №1):**
  * Вход `DI` на MAX490 "Шаг Z" ◄— Teensy **Pin 2**
  * Вход `DI` на MAX490 "Направление Z" ◄— Teensy **Pin 3**
  * **Синий провод** (SON) ◄— ПЛК **DO7** (релейный выход, sink к общему)
* **Ось C (Кабель LAN №2):**
  * Вход `DI` на MAX490 "Шаг C" ◄— Teensy **Pin 4**
  * Вход `DI` на MAX490 "Направление C" ◄— Teensy **Pin 5**
  * **Синий провод** (SON) ◄— ПЛК **DO8** (релейный выход, sink к общему)

## 5. Схема подключения входов CN1 (24В цепи)

### 5.1. Source mode (PNP)

> **ВАЖНО:** Внутри драйвера MR-JE-A на этих входах уже установлены резисторы ~6 кОм.
> Подключение идёт **напрямую к 24В/25В без внешних резисторов**.
> Добавление внешних резисторов снизит ток до ~2.9 мА — недостаточно для оптопар (требуется ~5 мА).

```
Внешний БП +24В (или +25В)
  │
  ├───────────[НЗ кнопка]──► DI0  (пин 19)   ← EMG (аварийный стоп, NC)
  ├───────────[НЗ конц.]────► LSP  (пин 43)   ← Концевик прямого хода (NC)
  ├───────────[НЗ конц.]────► LSN  (пин 44)   ← Концевик обратного хода (NC)
  ├───[ПЛК DO7/DO8 реле]──► SON  (пин 15)   ← Servo ON (sink/NPN)
  │
GND (внешний БП)
  │
  └────────────────────────► DICOM (пин 20 и 21) ← Общий входов (Source mode)
```

### 5.2. Sink mode (NPN) — рекомендуется для текущей схемы SON

```
Внешний БП +24В
  │
  └────────────────────────► DICOM (пин 20/21) = +24В

Вход активируется замыканием пина на GND:
  DI0 (пин 19) ──[ключ]──► GND
  SON (пин 15) ──[ключ]──► GND
  LSP (пин 43) ──[ключ]──► GND
  LSN (пин 44) ──[ключ]──► GND
```

### 5.3. Проверка допустимости прямого подключения при 25В

| Параметр | Значение | Допуск | Результат |
|----------|----------|--------|-----------|
| Напряжение питания | 25В | 12–26.4В (мануал) | ✓ в пределах |
| Ток через sink-вход (Source mode) | 11.85 мА | 5–20 мА | ✓ норма |
| Мощность на внутреннем резисторе | 0.28 Вт | ≤0.25 Вт (типовой) | ⚠ рекомендуются резисторы 0.5 Вт |

## 6. Подключение импульсных входов Step/Dir (RS-422, 5В)

### 6.1. Пины Step/Dir

| Функция | Пин (+) | Пин (−) | Стандарт |
|---------|---------|---------|----------|
| **STEP (импульсы)** | **10 (PP)** | **11 (PG)** | RS-422 (5В дифференциальный) |
| **DIR (направление)** | **35 (NP)** | **36 (NG)** | RS-422 (5В дифференциальный) |

> **⛔ КРИТИЧЕСКИ:** Запрещена подача 24/25В на пины 10, 11, 35, 36.
> Входы рассчитаны строго на **5В стандарт RS-422**.
> Подача большего напряжения приведёт к выгоранию внутренних оптопар!

### 6.2. Схема подключения Teensy 4.1 → MR-JE-200A через MAX490

```
Teensy 4.1 (3.3В)                     MR-JE-200A CN1
─────────────────                      ──────────────
                                        (50-пиновый разъём)

GPIO Step ──► [MAX490 TX+] ──► PP  (пин 10)   STEP +
               [MAX490 TX−] ──► PG  (пин 11)   STEP −

GPIO Dir  ──► [MAX490 TX+] ──► NP  (пин 35)   DIR +
               [MAX490 TX−] ──► NG  (пин 36)   DIR −
```

> Использование MAX490 (RS-422 Line Driver) **ОБЯЗАТЕЛЬНО** для дифференциальных входов.
> На текущем этапе (Teensy не поставлен) пины 10, 11, 35, 36 **НЕ ПОДКЛЮЧЕНЫ**.

## 10. Безопасность

### 9.1. Вход DI0 / EMG (аварийный стоп)

- Пин 19 (DI0) по умолчанию настроен как **EMG** (параметр PD12).
- EMG — вход **нормально замкнутого** (НЗ / NC) типа.
- Рабочее состояние: цепь замкнута → ток постоянно → серво разрешено.
- Нажатие аварийной кнопки: цепь разомкнута → ток прекращается → аварийная остановка.
- **Принцип безопасного отказа**: обрыв провода = остановка.

### 9.2. Концевики LSP / LSN (пины 43/44)

- НЗ контакты — обрыв провода = остановка (безопасный отказ).
- Для теста: перемычки (замыкание цепи). Для эксплуатации: НЗ индуктивные датчики.

### 9.3. Общие рекомендации по кабелю

- Экранированные кабели для CN1, особенно для Step/Dir (пины 10, 11, 35, 36).
- Экран заземлять с одной стороны (со стороны драйвера).
- Длина кабеля CN1 — не более 5 м.
- Разнести сигнальные и силовые (U/V/W) кабели минимум на 30 см.

## 11. Тестовая конфигурация (без Teensy, режим JOG)

```
Минимальное подключение CN1 для JOG Mode через MR Configurator2:

  +25В ──────────► SON   (пин 15)   Servo ON
  +25В ──────────► DI0   (пин 19)   EMG (или перемычка)
  +25В ──────────► LSP   (пин 43)   Концевик + (или перемычка)
  +25В ──────────► LSN   (пин 44)   Концевик − (или перемычка)
  GND  ──────────► DICOM (пин 20)   Общий входов (Source mode)

  Пины 10, 11, 35, 36 (Step/Dir) — НЕ ПОДКЛЮЧАТЬ.
```

При корректном подключении: дисплей `d01` → `d09` (серво ON, готов).

> `Pr.PD01 = 1000` устраняет AL.E6.1 (подтверждено).

## 12. Диагностика аварий MR-JE-200A

| Код | Описание | Решение |
|-----|----------|---------|
| AL.10 | Пониженное напряжение | Проверить питание L1/L2 |
| AL.16 | Ошибка энкодера | Проверить CN2 кабель/распиновку |
| AL.19 | Перегрузка по току | Проверить U/V/W, нагрузку |
| AL.32 | Превышение ошибки позиции | Снизить ускорение |
| AL.37 | Ошибка EMG | Подключить EMG на +24В (НЗ) |
| **AL.E6 / AL.E6.1** | **Серво выкл. при движении / некорректный порядок** | **Подать SON до импульсов, Pr.PD01=1000** |

**Как убрать AL.E6.1:**
1. Убрать тестовые импульсы с CN1 (STEP/DIR временно отключить)
2. Проверить: DICOM (pin 20/21) = 0V, DI0 (pin 19) = +24V, LSP (pin 43) = +24V, LSN (pin 44) = +24V, SON (pin 15) = +24V
3. Сбросить аварию (кнопка SET/RESET или через MR Configurator2)
4. Убедиться в `d09` перед подачей STEP
